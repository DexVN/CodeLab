HTML Images

The HTML img tag is used to embed an image in a web page.

The required src attribute specifies the path (URL) to the image.

The required alt attribute provides an alternate text for an image , is the user for some reason cannot view it (because of slow connection, an error in the src attribute, or if the user uses a screen reader).
