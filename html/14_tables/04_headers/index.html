Table Headers

Use the TH element to define table headers.

Firstname Lastname Age
Jill Smith 50
Eve Jackson 94

Vertical Table headers

The first colmn becomes table heaers if you set the first table cell in each table row to a TH element:

Fistname Jill Eve
Lastname Smith Jackson
Age 50 94

Left-align Headers

To left-align the table headers, se the CSS text-align property.

Firstname Lastname Age
Jill Smith 50
Eve Jackson 94

A header that spans two columns

Use the colspan attribute to have a header span over multiple columns.

Name Age
Jill Smith 50
Eve Jackson 94
