Some programing code:

The kbd Element

The kbd element is used to define keyboard input:

Save the document by pressing Ctr + S

The samp Element

The samp element is used to define sample output from a computer program.

Message from my computer:

File not found.
Press F1 to continue

The code element does not preserve whitespace and line-breaks.

To fix this, you can put the code element inside a pre element:

			
				x = 5;
				y =  6;
				z = x + y;

The var Element

The are of a triangle is: 1/2 xb x h, where b is the base, and h is the vertical height.
